Learning About Formats

What Are Formats?

Formats are the styles which are applied to text to make them look beautiful or to remind someone for something important.

Formatting In HTML.

Here are the different types of formats and their HTML tags:
<b>: Bold text.
<strong>: Important text.
<i>: Italic text.
<u>: Underlined text.
<em>: Emphasized text.
<mark>: Marked text.
<small>: Smaller text.
<del>: Deleted text.
<ins>: Inserted text.
<sub>: Subscripted text.
<sup>: Superscripted text.
Each tag should start with <...>, then the text to be mdified with that tag, then end with </...>
